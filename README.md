# 🛡️ Защити техстек проекта

Игра-конструктор: собери техстек проекта, выбирая по одной технологии в каждой категории.

## Правила

- Слева — карточки технологий (вразнобой, со случайным наклоном).
- У каждой карточки есть категории (например, `frontend`, `runtime`, `ORM`) и кнопка раскрытия описания.
- Выбираешь технологию → остальные карточки из тех же категорий **затеняются** (нельзя одновременно взять Next.js и Nuxt — они оба `meta-framework`).
- Справа — выбранный стек, сгруппированный по слоям: Языки / Фронтенд / Бэкенд / Данные / Тулинг / DevOps / AI.
- Выбор **сохраняется в localStorage** и переживает обновление страницы.
- Кнопка `🎲 перемешать` тасует карточки. Кнопка `сброс` чистит стек.

## Стек

- Vite + React 19 + TypeScript
- Tailwind CSS 3
- localStorage для сохранения

## Запуск

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # прод-сборка в ./dist
```

## Структура

```
src/
├── App.tsx                          # двухпанельный layout
├── data/technologies.ts             # каталог технологий
├── components/
│   ├── scattered-field.tsx          # левая панель — вразнобой
│   ├── tech-card.tsx                # карточка с раскрытием
│   └── stack-panel.tsx              # правая панель — стек по слоям
├── store/use-stack-store.ts         # выбор + локалсторадж + блокировка категорий
└── types.ts                         # категории, слои, метки
```

## Как расширить каталог

Добавь объект в `src/data/technologies.ts`:

```ts
{
  id: 'qwik',
  name: 'Qwik',
  emoji: '⚡',
  color: '#ac7ef4',
  categories: ['meta-framework'],
  layer: 'frontend',
  description: 'Resumable framework без гидрации.',
}
```

Если нужна новая категория или слой — добавь её в `src/types.ts` (в `CategoryId` / `Layer` + соответствующие `LABEL`-мапы).
